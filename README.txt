--------------------------------------------------------------------------------
Description
--------------------------------------------------------------------------------

This course format displays each section as a masonry brick. Using YUI 
(http://yui-masonry.appspot.com/), sections are dynamicly layouted into a 
fluent style page with colored bricks. 

--------------------------------------------------------------------------------
Dependencies
--------------------------------------------------------------------------------

Course Format topics

--------------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------------

1.	copy the folder named "masonry" into the moodle/course/format folder.
2.	Login to Moodle with admin rights. 
3.	The notification for the installation of the new course format is shown
4.     Click on Install
5.     Decide which default backcolor you want to use
